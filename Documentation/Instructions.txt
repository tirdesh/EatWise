Instructions To Run The Project: "EatWise - Nourishing Nutrition Management for Healthier Lifestyles!"

Navigate to the "EatWise" directory and locate the "view_controller.fx.FoodLoggerGUI" file.

Ensure that you have the JavaFX SDK (version 21.0.1) installed on your system.

Add the external JARs listed in the "javafx-sdk-21.0.1" folder to your Java Build Path under Preferences -> Java -> Build Path -> User Libraries.

After successfully configuring the project, run the view_controller.fx.FoodLoggerGUI file to launch the application.
